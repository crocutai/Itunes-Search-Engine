This is a [Next.js](https://nextjs.org) project allow users to search songs using iTunes Search API, provide the ability to filter and sort songs.

## Getting Started

First, run the development server:

```bash
# clone repository
git clone https://github.com/crocutai/Itunes-Search-Engine.git

# install package
npm install

# run dev 
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Start the Server
``` bash
# build the release 
npm run build

# start the server
npm run start
```
